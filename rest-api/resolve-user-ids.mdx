## `POST` `/webcast/resolve_user_ids`

Convert numeric TikTok user IDs to usernames.

### Authentication

Pass your API key as a query parameter:

```
/webcast/resolve_user_ids?apiKey=YOUR_API_KEY
```

### Example

<CodeGroup>
```javascript Node.js
const res = await fetch('https://api.tik.tools/webcast/resolve_user_ids?apiKey=YOUR_KEY', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ unique_id: 'username' })
});
const data = await res.json();
console.log(data);
```

```python Python
import requests
res = requests.post('https://api.tik.tools/webcast/resolve_user_ids',
    params={'apiKey': 'YOUR_KEY'},
    json={'unique_id': 'username'})
print(res.json())
```

```bash cURL
curl -X POST \"https://api.tik.tools/webcast/resolve_user_ids?apiKey=YOUR_KEY\" \
  -H "Content-Type: application/json" \
  -d '{"unique_id": "username"}'
```
</CodeGroup>
