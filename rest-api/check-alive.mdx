## `GET` `/webcast/check_alive`

Check if one or more TikTok users are currently live streaming.

### Authentication

Pass your API key as a query parameter:

```
/webcast/check_alive?apiKey=YOUR_API_KEY
```

### Example

<CodeGroup>
```javascript Node.js
const res = await fetch('https://api.tik.tools/webcast/check_alive?apiKey=YOUR_KEY');
const data = await res.json();
console.log(data);
```

```python Python
import requests
res = requests.get('https://api.tik.tools/webcast/check_alive',
    params={'apiKey': 'YOUR_KEY'})
print(res.json())
```

```bash cURL
curl \"https://api.tik.tools/webcast/check_alive?apiKey=YOUR_KEY\"
```
</CodeGroup>
